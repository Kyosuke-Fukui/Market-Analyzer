1.Herokuで新規アカウントを作成
https://www.heroku.com/
SignUpからで新規アカウントを作成します。

2.ログイン後、クレジットカード情報を登録（形式だけです）
右上のアイコン→Accountsetting→Billing→BillingInfomationでクレジットカード情報を登録
↑
clearDBをインストール際に必要（インストール後利用するのは無料プランのため課金はされない（上限5MB））

3.Heroku CLIをPCにインストール（不要かも）

4.Herokuで新規アプリを作成します

（5、6はDBの設定してからでOK）
5.作成したアプリの管理ページ（https://dashboard.heroku.com/apps/アプリ名/）
の「Deploy」→「Deployment method」で「Github」を選択
→リポジトリやbranchを選択し、デプロイ

6.「Setting」→「Domains」でアドレスが確認できる

（この時点ではDBの設定を何もしていないので当然CRUD処理などは動かない）

7.MySQL Workbench(DBをGUIで管理)のインストールと新規DBの作成

「Settings」→「Config Vars」の以下の記載を参照
mysql://b351165803b4f5:dac35273@us-cdbr-east-03.cleardb.com/heroku_60bde36f55eb52c?reconnect=true

CLEARDB_DATABASE_URL = mysql://ユーザ名:パスワード@ホスト名/DB名?reconnect=true
Connection Name ： 任意の名前
Hostname ： ホスト名
Username ： ユーザ名
Password ： パスワード
Default Schema ： DB名

8.phpMyAdminからデータをエクスポート
@DATABASE→@@SERVER@にしてエクスポート
（DBから複数テーブル選択してまとめてエクスポートも可）

9.MySQL Workbenchにインポート
Start Importが画面上表示されていなかったのでTabキー使って頑張る

10.DB接続部分のコードを以下に書き換え

$db = parse_url($_SERVER['CLEARDB_DATABASE_URL']);
  $db['dbname'] = ltrim($db['path'], '/');
  $dsn = "mysql:host={$db['host']};dbname={$db['dbname']};charset=utf8";
  $user = $db['user'];
  $password = $db['pass'];
  $options = array(
    PDO::ATTR_ERRMODE => PDO::ERRMODE_EXCEPTION,
    PDO::ATTR_DEFAULT_FETCH_MODE => PDO::FETCH_ASSOC,
    PDO::MYSQL_ATTR_USE_BUFFERED_QUERY =>true,
  );
  $dbh = new PDO($dsn,$user,$password,$options);
